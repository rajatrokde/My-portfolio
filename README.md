# Add Jenkins Credentials (Recommended) #

In Jenkins → Manage Jenkins → Credentials → System → Global credentials

Add Username with password:

Username = Docker Hub username

Password = Docker Hub access token (better than password)

Use credentialsId in the pipeline (docker-hub-creds in the example above)
